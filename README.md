Breast Cancer Detection using Machine Learning

This project uses Machine Learning to detect whether a tumor is Malignant (Cancerous) or Benign (Non-Cancerous) using the Breast Cancer dataset.

**Objective**
Build a model to predict breast cancer
Understand Machine Learning in healthcare
**Dataset**
Breast Cancer Wisconsin Dataset
Loaded using sklearn
**Technologies Used**
Python
Scikit-learn
NumPy
Pandas
**Model**
Logistic Regression
